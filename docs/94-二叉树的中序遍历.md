# 94. 二叉树的中序遍历

## 题目描述

给定一个二叉树的根节点 `root`，返回它的 **中序遍历** 。

## 解题思路

### 方法 1: 递归（推荐）

最直观的方法，按照"左 → 根 → 右"的顺序递归遍历。

### 方法 2: 迭代

使用栈模拟递归过程，避免递归开销。

### 方法 3: Morris 遍历

在不使用额外空间的情况下实现中序遍历。

## 代码实现

```python
class Solution:
    # 方法 1: 递归
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        result = []
        
        def dfs(node):
            if not node:
                return
            dfs(node.left)
            result.append(node.val)
            dfs(node.right)
        
        dfs(root)
        return result
    
    # 方法 2: 迭代
    def inorderTraversal_iterative(self, root: TreeNode) -> List[int]:
        result = []
        stack = []
        current = root
        
        while current or stack:
            # 遍历到最左的节点
            while current:
                stack.append(current)
                current = current.left
            
            # 当前为空，弹出栈顶
            current = stack.pop()
            result.append(current.val)
            
            # 遍历右子树
            current = current.right
        
        return result
```

## 复杂度分析

- **时间复杂度**：$O(n)$，各方法都需遍历所有节点
- **空间复杂度**：$O(h)$，其中 $h$ 是树的高度
