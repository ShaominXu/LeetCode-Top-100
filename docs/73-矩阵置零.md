# 73. 矩阵置零

## 题目描述

给定一个 `m x n` 的矩阵，如果一个元素为 0 ，则将其所在行与列的所有元素都设为 0 。请在 **原地** 修改输入矩阵。

## 解题思路

### 方法 1: 使用第一行第一列作为标记（推荐）

利用矩阵的第一行和第一列存储哪些行列需要置零，避免额外空间。需要特殊处理第一行和第一列本身是否需要置零。

### 方法 2: 标记集合

记录哪些行和列需要置零，然后再次遍历矩阵进行置零。

## 代码实现

```python
class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        m, n = len(matrix), len(matrix[0])
        first_row_zero = False
        first_col_zero = False
        
        # 检查第一行和第一列是否需要置零
        for i in range(m):
            if matrix[i][0] == 0:
                first_col_zero = True
                break
        
        for j in range(n):
            if matrix[0][j] == 0:
                first_row_zero = True
                break
        
        # 使用第一行第一列作为标记
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][j] == 0:
                    matrix[i][0] = 0
                    matrix[0][j] = 0
        
        # 根据标记置零
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][0] == 0 or matrix[0][j] == 0:
                    matrix[i][j] = 0
        
        # 处理第一行
        if first_row_zero:
            for j in range(n):
                matrix[0][j] = 0
        
        # 处理第一列
        if first_col_zero:
            for i in range(m):
                matrix[i][0] = 0
```

## 复杂度分析

- **时间复杂度**：$O(m \times n)$，需要遍历矩阵多次
- **空间复杂度**：$O(1)$，仅使用常数级额外变量
