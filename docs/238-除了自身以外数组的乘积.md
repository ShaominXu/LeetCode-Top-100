# 238. 除了自身以外数组的乘积

## 题目描述

给你一个整数数组 `nums`，返回 *数组 `answer`* ，其中 `answer[i]` 等于 `nums` 中除 `nums[i]` 之外其余各元素的乘积。

题目数据 **保证** 数组之中任意元素的全前缀元素和全后缀元素的乘积都在  **32 位** 整数范围内。

**不能使用除法**，且在 **O(n) 时间复杂度**内完成此题。

## 解题思路

### 方法 1: 前缀和后缀乘积（推荐）

- 先计算每个位置之前所有元素的乘积（前缀）
- 再计算每个位置之后所有元素的乘积（后缀）
- 两者相乘得到答案

### 方法 2: 单次前缀遍历

优化空间，只用一个额外数组。

## 代码实现

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        answer = [1] * n
        
        # 计算前缀乘积
        prefix = 1
        for i in range(n):
            answer[i] = prefix
            prefix *= nums[i]
        
        # 计算后缀乘积并与前缀相乘
        suffix = 1
        for i in range(n - 1, -1, -1):
            answer[i] *= suffix
            suffix *= nums[i]
        
        return answer
```

## 复杂度分析

- **时间复杂度**：$O(n)$，遍历数组两次
- **空间复杂度**：$O(1)$，不计返回数组的空间
