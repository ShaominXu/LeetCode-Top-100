# 双指针

双指针（Two Pointers）是一种常用的编程技巧，在处理数组、链表和字符串等顺序结构时特别有效。

## 核心思想

使用两个指针分别指向序列的不同位置，通过同向移动或相向移动来解决问题。

## 主要应用场景

1. **相向双指针**：两个指针从两端向中间移动
   - 二分搜索
   - 容器盛水问题
   - 三数之和

2. **同向双指针**：两个指针按同一方向移动
   - 滑动窗口
   - 快慢指针
   - 数组去重、分类

3. **背向双指针**：两个指针从中间向两端移动
   - 回文串扩展
   - 中心扩展算法

## 双指针的优势

- ✨ **时间高效**：通常能将时间复杂度从 O(n²) 降低到 O(n)
- 💾 **空间优化**：大多数情况下只需 O(1) 额外空间
- 📝 **代码简洁**：相比暴力方法更直观紧凑

## 常见套路

- 排序数组 + 双指针求和问题
- 快慢指针寻找链表中点
- 滑动窗口求子数组/子字符串
