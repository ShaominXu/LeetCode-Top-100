# 51. N 皇后

## 题目描述

N 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

给定一个整数 `n`，返回所有不同的 **n 皇后问题** 的解决方案。

每一种解法包含一个不同的 **n 皇后问题** 的棋盘排列，该排列中 `'Q'` 和 `'.'` 分别代表皇后和空位。

## 解题思路

### 方法 1: 回溯法（推荐）

使用回溯搜索所有可能的棋盘配置。在放置皇后前检查是否冲突：
- 同列是否有皇后
- 左上-右下对角线是否有皇后  
- 右上-左下对角线是否有皇后

### 方法 2: 集合优化

使用集合记录已被占据的列和两条对角线，避免重复检查。

## 代码实现

```python
class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        result = []
        board = ['.' * n for _ in range(n)]
        cols = set()
        diag1 = set()  # row - col
        diag2 = set()  # row + col
        
        def backtrack(row):
            if row == n:
                result.append(board[:])
                return
            
            for col in range(n):
                if col in cols or row - col in diag1 or row + col in diag2:
                    continue
                
                # 放置皇后
                board[row] = '.' * col + 'Q' + '.' * (n - col - 1)
                cols.add(col)
                diag1.add(row - col)
                diag2.add(row + col)
                
                backtrack(row + 1)
                
                # 回溯
                board[row] = '.' * n
                cols.remove(col)
                diag1.remove(row - col)
                diag2.remove(row + col)
        
        backtrack(0)
        return result
```

## 复杂度分析

- **时间复杂度**：$O(N!)$，最坏情况下需要尝试所有排列
- **空间复杂度**：$O(N)$，递归栈深度和集合空间
