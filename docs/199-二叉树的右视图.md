# 199. 二叉树的右视图

## 题目描述

给定一个二叉树的 **根节点** `root`，想象自己站在树的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

## 解题思路

### 方法 1: 层序遍历（推荐）

对每一层进行 BFS 遍历，记录每层的最后一个节点（最右侧的节点）。

### 方法 2: DFS + 右子树优先

使用 DFS 遍历，优先遍历右子树，记录每层首次访问的节点。

## 代码实现

```python
class Solution:
    # 方法 1: 层序遍历
    def rightSideView(self, root: TreeNode) -> List[int]:
        if not root:
            return []
        
        result = []
        queue = [root]
        
        while queue:
            result.append(queue[-1].val)
            
            next_queue = []
            for node in queue:
                if node.left:
                    next_queue.append(node.left)
                if node.right:
                    next_queue.append(node.right)
            
            queue = next_queue
        
        return result
    
    # 方法 2: DFS
    def rightSideView_dfs(self, root: TreeNode) -> List[int]:
        result = []
        
        def dfs(node, depth):
            if not node:
                return
            
            # 首次访问该深度时记录
            if depth == len(result):
                result.append(node.val)
            
            # 优先遍历右子树
            dfs(node.right, depth + 1)
            dfs(node.left, depth + 1)
        
        dfs(root, 0)
        return result
```

## 复杂度分析

- **时间复杂度**：$O(n)$，需遍历所有节点
- **空间复杂度**：$O(h)$，其中 $h$ 是树的高度
