# 118. 杨辉三角

## 题目描述

给定一个非负整数 `numRows`，生成「杨辉三角」的前 `numRows` 行。

在「杨辉三角」中，每个数是它左上方和右上方的数之和。

## 解题思路

### 方法 1: 动态规划（推荐）

每一行的首尾都是 1，中间的元素等于上一行相对应的两个元素之和。

### 方法 2: 数学公式

利用组合数公式计算每个元素的值。

## 代码实现

```python
class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        result = []
        
        for i in range(numRows):
            row = [1] * (i + 1)
            
            # 计算中间的元素
            for j in range(1, i):
                row[j] = result[i - 1][j - 1] + result[i - 1][j]
            
            result.append(row)
        
        return result
```

## 复杂度分析

- **时间复杂度**：$O(n^2)$，共有 $\frac{n(n+1)}{2}$ 个元素
- **空间复杂度**：$O(1)$，不计返回结果的空间
